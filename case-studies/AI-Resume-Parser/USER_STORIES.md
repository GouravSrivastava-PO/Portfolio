\# User Stories



\## Epic: Resume Upload and Parsing



\### User Story 1



\*\*As a Recruiter\*\*



I want to upload a candidate resume



So that the system can automatically extract candidate information.



\### Acceptance Criteria



\* User can upload PDF resumes

\* User can upload DOCX resumes

\* File validation is performed

\* Upload confirmation is displayed



\---



\### User Story 2



\*\*As a Recruiter\*\*



I want candidate information extracted automatically



So that I do not have to manually review every resume.



\### Acceptance Criteria



\* Name extracted

\* Skills extracted

\* Experience extracted

\* Education extracted



\---



\### User Story 3



\*\*As a Hiring Manager\*\*



I want candidates scored against job requirements



So that I can prioritize relevant candidates.



\### Acceptance Criteria



\* Match score generated

\* Skills comparison available

\* Experience comparison available

\* Ranking displayed



\---



\### User Story 4



\*\*As a Recruiter\*\*



I want an AI-generated candidate summary



So that I can quickly understand profile suitability.



\### Acceptance Criteria



\* Summary generated automatically

\* Key strengths highlighted

\* Relevant experience highlighted

\* Summary available within seconds
